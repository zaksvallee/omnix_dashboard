MSVALLEE face gallery naming

Use folder names under this site gallery to classify recognized people:

- `MSVALLEE_RESIDENT_*` for known residents. These matches suppress direct alerts.
- `MSVALLEE_VISITOR_*` for known visitors. These matches stay known but are not treated as unknown people.
- `MSVALLEE_FLAGGED_*` for watchlist or banned people. These matches escalate immediately as high-priority alerts.

Example:

- `MSVALLEE_FLAGGED_EXAMPLE`

Put one or more clear face photos for the flagged individual in this folder.
